# Task Manager - Agent Instructions

## Project Overview
A Qt6 C++17 desktop Task Manager application for Linux that displays live system process information in a GUI table.

## Agent Instructions
* **Operating System:** The system environment is Arch Linux (not Debian). Ensure any OS-specific commands or paths reflect this.
* **Version Control:** Always create a Git branch with an appropriate name before creating new files or modifying existing files.
* **Compilation & Testing:** Do not compile the project or run tests. The user will handle compilation and testing manually.
* **C++ Standards:** Always use the `cpp-coding-standards` skill before writing any C++ code.
* **Documentation:** Always refer to `context7` for any documentation.
* **UI/UX Design:** When designing the Qt6 app layout, always refer to the `qt6-desktop-ux` skill.
